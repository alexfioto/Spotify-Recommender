Hi! I'm Alex Fioto, Data Scientist and Professional Musician.

I created this app to find better song recommendations on Spotify. This site was created using a Flask framework and deployed on Heroku. I'm using Pandas and scikit-learn maching learning library on the backend to help find rockin' jams. A big thanks to Spotify for a stellar API and to SpotiPY for creating a wonderful python wrapper. Check out my GitHub Repo for more information!

Choose an artist and generate a playlist of their songs that match your taste!

"Hey, you should check out this cool new band called Dinosaur Pile-Up!" I get this a lot from friends and family. It's difficult to know where to start. This recommender will pull all tracks from a given artist and compare them to top tracks in your listening history. A playlist will be created with songs that match your taste.
